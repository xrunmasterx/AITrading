# 🚀 AITrading 启动方式

## 方式一：双击批处理文件（推荐）

**最简单的方式**：双击 `启动应用.bat` 文件

- ✅ 无需输入命令
- ✅ 自动打开浏览器
- ✅ 显示启动日志

## 方式二：PowerShell 脚本（推荐）

双击 `启动应用.ps1` 文件

**特点**：
- ✅ 自动检查 Python 环境
- ✅ 自动激活虚拟环境（如果存在）
- ✅ 自动检查依赖
- ✅ 彩色输出，更友好

**如果提示"无法运行脚本"**：
1. 右键点击 `启动应用.ps1`
2. 选择"属性"
3. 点击"解除锁定"
4. 或者在 PowerShell 中运行：
   ```powershell
   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
   ```

## 方式三：静默启动（无命令行窗口）

双击 `启动应用-静默.vbs` 文件

- ✅ 后台运行，不显示命令行窗口
- ✅ 适合创建桌面快捷方式

## 方式四：创建桌面快捷方式

### Windows 10/11：

1. 右键点击 `启动应用.bat` 或 `启动应用.ps1`
2. 选择"发送到" → "桌面快捷方式"
3. 右键点击桌面快捷方式 → "属性"
4. 可以更改图标（可选）：
   - 点击"更改图标"
   - 选择 `C:\Windows\System32\shell32.dll` 中的图标
5. 点击"确定"

### 或者手动创建：

1. 在桌面右键 → "新建" → "快捷方式"
2. 输入位置：
   ```
   E:\AITrading\启动应用.bat
   ```
   （替换为您的实际路径）
3. 点击"下一步"，输入名称：`AITrading`
4. 点击"完成"

## 方式五：任务栏固定

1. 启动应用后，在任务栏找到 Streamlit 图标
2. 右键点击 → "固定到任务栏"
3. 下次可以直接从任务栏启动

## 停止应用

- 在命令行窗口按 `Ctrl + C`
- 或者关闭浏览器标签页后，命令行会自动停止

## 常见问题

### Q: 双击后没有反应？
A: 检查是否安装了 Python，以及是否在项目目录下

### Q: 提示"找不到 python"？
A: 确保 Python 已添加到系统 PATH，或使用完整路径

### Q: 端口被占用？
A: 修改启动脚本中的端口号，或关闭占用 8501 端口的程序

### Q: 想要自定义端口？
A: 编辑启动脚本，修改 `--server.port 8501` 为其他端口
